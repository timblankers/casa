# Server

NodeJS API using Express.

## Prerequisites
- NodeJS
- NPM
- WiringX https://github.com/wiringX/wiringX
- KaKu binary for GPIO ports

## Installation
```
$ npm install
```

## Run
```
$ npm start
```
